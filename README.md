# 🍳 Світ Рецептів (Recipes World) — Full-Stack Web App

Веб-додаток для керування кулінарними рецептами, створений у межах вивчення сучасних веб-технологій. Проєкт поєднує в собі динамічний інтерфейс на React та власний серверний бекенд.

## 🚀 Живе посилання (Live Demo)
Ви можете переглянути працюючий додаток за адресою:  
👉 **[https://ann-y-06.github.io](https://ann-y-06.github.io)**



## 🛠 Технологічний стек

### **Frontend:**
* **React 18** — побудова компонентного інтерфейсу.
* **React Router** — навігація між сторінками (Рецепти, Категорії, Коментарі).
* **CSS3** — адаптивний дизайн та стилізація.

### **Backend:**
* **Node.js & Express** — створення REST API для обробки запитів.
* **Firebase Admin SDK** — безпечна взаємодія з базою даних.
* **Render.com** — хмарний хостинг для серверної частини.

### **Database:**
* **Firebase Realtime Database** — хмарне збереження даних у форматі NoSQL.



## 📋 Функціонал
* **Перегляд рецептів:** Автоматичне завантаження даних з бекенду при відкритті сторінки.
* **Додавання та редагування:** Можливість створювати нові записи або змінювати існуючі через зручну форму.
* **Видалення:** Повне видалення рецептів з бази даних через API.
* **Категорії:** Фільтрація страв за типами.
* **Авторизація:** Система входу через Firebase Auth для ідентифікації авторів рецептів.
* **Коментарі:** Можливість залишати відгуки до страв у реальному часі.



## 🏗 Архітектура проєкту
Проєкт реалізовано за клієнт-серверною моделлю:
1. Клієнт (**GitHub Pages**) надсилає HTTP-запити (`GET`, `POST`, `DELETE`).
2. Сервер (**Render**) приймає запити, перевіряє права доступу та звертається до бази.
3. База даних (**Firebase**) зберігає та повертає JSON-дані.
